Description -

This site is a basic way to track items obtained in the video game Destiny 2 made by the company Bungie. I'm trying to recreate a website that was made for the original Destiny game at www.destinyexotics.com which looks like this by using AngularJS which I wanted to become familiar with before first using it at work. It was also deployed previously as a C# web application on an AWS EBS instance, but it didnt need to stay there as it's now completely in Angular with no C# left. The data is populated by a Python script seen here which grabs from the Destiny API. I had originally planned to make the API calls in C#, but found it was significantly easier to unpack the Destiny SQLite DB with Python.
